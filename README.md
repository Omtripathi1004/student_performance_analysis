# OBJECTIVE :- this project explores the factors that influnce student academy using python and seaborn.

1. Analyze student performance data.
2. Identify factors affecting exam scores.
3. Examine relationships between study hours, attendance, and academic achievement.
4. Detect patterns, trends, and outliers in student data.
5. Generate actionable insights through visualizations.

# Tools & Technologies:-
1. Python
2. Pandas
3. NumPy
4. Matplotlib
5. Seaborn
6. Vs code

# Data Preprocessing:-

The following steps were performed:

1. Checked for missing values.
2. Removed duplicate records.
3. Verified data types.
4. Cleaned and prepared data for analysis.

# Key Findings:-

Students who studied more hours generally achieved higher exam scores.
Attendance showed a positive relationship with academic performance.
Previous academic scores were strong indicators of future performance.
Some outliers were identified in study hours and exam scores.
Multiple factors collectively influenced student success.

# Conclusion:-

This project demonstrates how data analytics can be used to understand the factors affecting student 
performance. Through visualization and statistical analysis, valuable insights were extracted that can 
help educators and students make informed decisions to improve academic outcomes.
